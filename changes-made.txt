================================================================================

PROMPT: "Can we append the session data to hubspot contacts with the API?"

CHANGES MADE:
1. Created HubSpot Service (lib/hubspotService.ts)
   - createOrUpdateHubSpotContact(): Main sync function
     * Searches for existing contact by email (avoids duplicates)
     * Creates new contact if none exists
     * Updates existing contact with new session data
     * Maps SessionData fields to HubSpot contact properties
     * Supports conversation transcript storage
     * Returns success status and contact ID
   
   - testHubSpotConnection(): Connection validation
     * Tests API key and permissions
     * Useful for debugging integration
   
   - Field Mapping:
     * Standard fields: email, firstname, lastname, phone, company, industry
     * Custom AIVI fields (must be created in HubSpot):
       - aivi_session_id: Unique session identifier
       - aivi_challenge: Customer's primary challenge
       - aivi_channels: Preferred communication channels (comma-separated)
       - aivi_volume: Monthly message volume
       - aivi_goal: Primary automation goal
       - aivi_current_crm: Current CRM system
       - aivi_conversation_transcript: Full chat transcript
       - aivi_additional_notes: Extra notes from form
   
   - Error Handling:
     * Validates API key configuration
     * Requires email for contact creation
     * Logs detailed errors for debugging
     * Detects missing custom properties

2. Created HubSpot Sync API Endpoint (app/api/hubspot/sync/route.ts)
   - POST /api/hubspot/sync
     * Accepts: sessionId, includeConversation, conversationTranscript
     * Retrieves session data from memory
     * Calls HubSpot service to create/update contact
     * Returns: success status, contactId, isNewContact flag
     * Error handling with detailed status codes
   
   - GET /api/hubspot/sync (test endpoint)
     * Returns API configuration status
     * Shows if API key is configured
     * Useful for health checks

3. Updated ChatBot Component (components/ChatBot.tsx)
   - Added syncToHubSpot() function
     * Checks if email exists before syncing
     * Optionally includes conversation transcript
     * Creates formatted timestamp for each message
     * Handles errors gracefully (doesn't block chat)
     * Logs all sync attempts for debugging
   
   - SYNC TRIGGER #1: Email Capture
